# fuzzy-text-watchface

A watchface for the Pebble Time 2 (port of [asellitt/fuzzy-text-clockface](https://github.com/asellitt/fuzzy-text-clockface)).

Available here: https://apps.repebble.com/a1aafb673792493195615f3c

Displays the current time using fuzzy natural language. Also shows heart rate, calories burned, steps and battery level
